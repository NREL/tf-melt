.. _tests-module:

Tests
=====

The tests module contains a set of unit tests for the TF-MELT package. These are not
intended to be exhaustive, but rather to provide a basic level of coverage for the
package for development purposes.


Submodules
----------

The tests module contains the following submodules:

- :ref:`Test Models <tests.test_models>`: Unit tests for the models module.
